Welcome to xMath API

The xMath API provides random mathematical expressions with various operations. Below are the available routes and their descriptions:

Routes

  • Random Expression: /api/random - Generates a random math expression.
  • Addition: /api/add - Generates a random addition expression.
  • Subtraction: /api/sub - Generates a random subtraction expression.
  • Multiplication: /api/mul - Generates a random multiplication expression.
  • Division: /api/div - Generates a random division expression.

Parameters

You can customize the API responses with the following parameters:

  • ?max: Sets the maximum number in the expression.
  • ?min: Sets the minimum number in the expression.
  • ?negative=1: Allows negative results in subtraction.

Example Usage

To get a random addition expression with numbers between 100 and 999:

https://x-math.azurewebsites.net/api/add?max=999&min=100
